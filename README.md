# Profit Pulse - Retail Sales Analytics Dashboard

## Overview

Profit Pulse is an end-to-end retail analytics project built using Python, SQL, and Power BI. The project analyzes retail transaction data to uncover customer behavior, sales trends, profitability metrics, and business insights.

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib)
* SQL
* Power BI
* Excel

## Project Workflow

1. Data Collection and Cleaning
2. Exploratory Data Analysis (EDA)
3. Customer Segmentation (RFM Analysis)
4. Sales and Profitability Analysis
5. Interactive Power BI Dashboard

## Key Insights

* Identified high-value customer segments using RFM analysis.
* Analyzed product and sales performance trends.
* Discovered revenue-driving customer groups.
* Built an interactive dashboard for business decision-making.

## Project Structure

* dashboard/ → Power BI dashboard files
* data/ → Raw dataset
* exports/ → Processed datasets
* notebooks/ → Data analysis notebooks
